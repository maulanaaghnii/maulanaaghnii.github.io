# Common .NET Packages

A list of essential NuGet packages for common tasks in .NET development.

## Data Access
- **Microsoft.EntityFrameworkCore**: Official ORM for .NET.
- **Dapper**: High-performance micro-ORM.
- **Npgsql.EntityFrameworkCore.PostgreSQL**: Provider for PostgreSQL.

## Utility & Mapping
- **AutoMapper**: Object-to-object mapping.
- **Newtonsoft.Json**: Classical JSON library.
- **FluentValidation**: Strongly-typed validation rules.

## Logging & Monitoring
- **Serilog**: Structured logging.
- **AppMetrics**: Performance metrics and reporting.

## Testing
- **xUnit**: Testing framework.
- **Moq**: Mocking library.
- **FluentAssertions**: Readable assertions.

## API & Documentation
- **Swashbuckle.AspNetCore**: Swagger/OpenAPI support.
- **MediatR**: Mediator pattern for CQRS.
