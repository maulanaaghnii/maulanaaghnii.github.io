# Ubuntu Essential Keyboard Shortcuts

Mastering keyboard shortcuts will significantly speed up your workflow in Ubuntu.

## General System
- `Super` (Windows Key): Open Activities search.
- `Super + A`: Show all applications.
- `Super + Tab`: Switch between running applications.
- `Super + L`: Lock the screen.
- `Alt + F2`: Open "Run Command" prompt.
- `PrtSc`: Take a screenshot.

## Window Management
- `Super + Page Up/Down`: Switch between workspaces.
- `Super + Left/Right Arrow`: Snap window to side (Split screen).
- `Super + Up Arrow`: Maximize window.
- `Super + Down Arrow`: Restore/Minimize window.
- `Alt + F4`: Close the current window.

## Terminal (GNOME Terminal)
- `Ctrl + Alt + T`: Open a new terminal.
- `Ctrl + Shift + T`: Open a new tab.
- `Ctrl + Shift + C`: Copy.
- `Ctrl + Shift + V`: Paste.
- `Ctrl + D`: Close the current tab/session.
- `Ctrl + L`: Clear the terminal screen.
