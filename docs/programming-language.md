# Programming Languages

A guide to the various programming languages covered in this wiki.

## Core Languages
- **[C#](csharp.md)**: The primary language for .NET development.
- **[Go (Golang)](golang-microservices.md)**: Excellent for performant microservices and infrastructure.
- **[Python](python.md)**: Great for scripting, data science, and automation.
- **JavaScript/TypeScript**: Essential for web development.

## Comparison Table

| Language | Paradigm | Typing | Common Use Cases |
|----------|----------|--------|------------------|
| C#       | Multi    | Strong | Web, Desktop, Games (Unity) |
| Go       | Impr     | Strong | Cloud, Microservices, CLI |
| Python   | Multi    | Dynamic| AI, Scripts, Web |
| Rust     | Multi    | Strong | Systems, WASM, Performance |

## Choosing a Language
- **Scale and Performance**: Go or C#.
- **Speed of Development**: Python or TypeScript.
- **System Level**: C++, Rust, or C.
- **Enterprise Web Apps**: Java or C#.
