# Tips and Tricks

General development tips and tricks to improve your workflow.

## General
- **Keyboard Shortcuts**: Learn the shortcuts for your IDE (VS Code, JetBrains).
- **Dotfiles**: Version control your configuration files.
- **Alias Everything**: Use bash/zsh aliases for frequent commands.

## Dotnet
- Use `dotnet watch` for hot-reload during development.
- Use `global.json` to pin your SDK version.

## Docker
- Use `.dockerignore` to keep your images small.
- Multi-stage builds are your friend for production images.

## Git
- Use `git commit --amend` to fix the last commit.
- `git checkout -` to switch back to the previous branch.
