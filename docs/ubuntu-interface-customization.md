# Ubuntu Interface Customization

Make your Ubuntu desktop look and feel exactly how you want.

## 1. GNOME Tweaks
The essential tool for deep customization.
```bash
sudo apt install gnome-tweaks gnome-shell-extension-manager -y
```

## 2. Appearance Features
- **Dark Mode**: Settings -> Appearance.
- **Accent Color**: Available in Ubuntu 22.04+.
- **Desktop Icons & Dock**: Adjust size and position in Settings.

## 3. Themes (Icons & GTK)
Find themes on [GNOME-Look](https://www.gnome-look.org/).
1. Download a theme.
2. Extract to `~/.themes` (for GUI) or `~/.icons` (for icons).
3. Select them in **Tweaks**.

## 4. Extensions
Enhance functionality with extensions from [extensions.gnome.org](https://extensions.gnome.org/).
- **Dash to Dock**: Transform the Ubuntu dock into a more flexible bar.
- **Just Perfection**: Cleanup the UI elements.
- **Blur my Shell**: Adds aesthetic blur effects.
