# NuGet Package Management

This topic is covered in detail on the **[Dotnet NuGet](dotnet-nuget.md)** page.

## Quick Summary
- **NuGet** is the package manager for .NET.
- Use `dotnet add package <Name>` to add libraries.
- Use `nuget.config` to manage private feeds.
- **CPM (Central Package Management)** is the modern way to manage versions across large solutions.
