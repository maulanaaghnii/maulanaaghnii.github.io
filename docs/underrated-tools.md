# Underrated Tools

A list of tools that are incredibly useful but often overlooked by developers.

## CLI Tools
- **[Fzf](https://github.com/junegunn/fzf)**: A general-purpose command-line fuzzy finder.
- **[Bat](https://github.com/sharkdp/bat)**: A cat clone with syntax highlighting and Git integration.
- **[Zoxide](https://github.com/ajeetdsouza/zoxide)**: A smarter `cd` command.
- **[Exa / Eza](https://github.com/eza-community/eza)**: A modern replacement for `ls`.
- **[Tldr](https://tldr.sh/)**: Simplified and community-driven man pages.

## GUI Tools
- **[DBeaver](https://dbeaver.io/)**: Universal database tool.
- **[Insomnia](https://insomnia.rest/)**: Excellent API client (alternative to Postman).
- **[Obsidian](https://obsidian.md/)**: A powerful knowledge base that works on top of a local folder of plain text Markdown files.

## Utilities
- **[Chocolatey](chocolatey.md)**: Package manager for Windows.
- **[WSL](wsl.md)**: Windows Subsystem for Linux.
