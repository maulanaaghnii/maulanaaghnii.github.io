## Essential Commands
```bash
# Update package list
sudo apt update

# Upgrade packages
sudo apt upgrade

# Install a package
sudo apt install curl git
```

## Useful Tools
- **WSL (Windows Subsystem for Linux)**: Run Ubuntu natively on Windows.
- **Docker**: Containerize applications easily on Ubuntu.
