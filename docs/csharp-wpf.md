# C# Windows Presentation Foundation (WPF)

WPF is a UI framework for creating desktop client applications.

## Key Features
- **XAML**: An XML-based language used to define the UI.
- **Data Binding**: Connect UI elements to data objects easily.
- **MVVM Pattern**: The recommended architectural pattern (**M**odel-**V**iew-**V**iew**M**odel).
- **Styling and Templating**: High degree of customization for controls.

## Basic XAML
```xml
<Button Content="Click Me" 
        HorizontalAlignment="Center" 
        VerticalAlignment="Center" 
        Click="Button_Click"/>
```

## Creating a Project
```bash
dotnet new wpf -n MyWpfApp
```
